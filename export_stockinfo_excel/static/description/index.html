Export Product Stock in Excel

Description

How to create Export Stock Report

-You can create export stock through the reporting of the Inventory in odoo.

Export Stock wizard

-After clicking on Export Stock, one wizard will appear, in that you have to configure dates for the report, warehouse(s), location(s) and other information.

Export Stock Report in Pdf

-PDF Report as per the configured warehouse(s), location(s) and date from (….) to (….).

Export Stock Report in Excel

-Excel Report as per the configured warehouse(s), location(s) and date from (….) to (….). (Negative stock will be highlighted in "Red" cells.).

How data appear in excel

-(each warehouse’s data stay inside its own table).
